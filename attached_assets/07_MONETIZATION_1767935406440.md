# 📂 STEP 7: 상업화 구조 (Monetization Strategy)

## 1. Core Philosophy: "Pay for Insight, Not for Win"
게임의 승패에 영향을 주는 아이템(공격력 강화 등)은 절대 팔지 않습니다. 이는 '지능 대결'이라는 게임의 본질을 해칩니다.
우리는 **"더 똑똑해질 수 있는 기회(Insight)"**와 **"쾌적한 연산 환경(Speed)"**을 팝니다.

* **Zero Server Cost Advantage:**
    * 경쟁사(ChatGPT API 기반 앱)는 유저가 늘어날수록 API 비용이 증가하여 구독료가 비쌉니다 ($10~$20/월).
    * **Move 37**은 유지비가 없으므로, **$2.99 ~ $9.99**의 파격적인 가격으로 시장을 파괴할 수 있습니다.

---

## 2. User Tier Segmentation (사용자 등급 구분)

### 🟢 Tier 1: The Human (Free User)
> *"도전을 즐기는 일반 사용자"*

* **Access:**
    * 모든 게임 모드 플레이 가능 (하지만 하루 플레이 횟수 제한).
    * 기본 AI 모델 (Standard Model) 사용.
    * 기본 스킨 제공.
* **Restriction (The Hook):**
    * **Neural Overheat (시스템 과부하):** 하루 5판 플레이 후, "AI 과열"로 인해 쿨타임 발생 (광고 시청 시 즉시 냉각).
    * **Black Boxed:** AI가 왜 그런 수를 두었는지 보여주는 **'Deep Analysis (심층 분석)' 기능 잠김.** 결과만 알 수 있고 이유는 모름.

### 🟣 Tier 2: The Transhuman (Pro User / One-time Purchase)
> *"논리와 분석을 원하는 헤비 유저"*

* **Price Strategy:** **$9.99 (Lifetime Unlock)** - 구독 피로도가 높은 요즘, '평생 소장'은 강력한 구매 유인이 됨.
* **Benefits:**
    * **Unlimited Energy:** 쿨타임/과부하 시스템 완전 제거.
    * **Glass Box Mode (핵심):** 게임 종료 후 AI의 **`<think>` 태그 전체 열람 가능.** AI의 사고 과정을 복기하며 학습 가능.
    * **Turbo Mode:** GPU 우선순위를 높여 연산 속도 2배 향상 (기술적 최적화 옵션 해금).
    * **Ad-Free:** 모든 광고 제거.

## 3. Detailed Revenue Streams (세부 수익원)

### A. Rewarded Ads (광고 - 보상형)
배너 광고(Banner Ads)는 앱의 품격을 떨어뜨리므로 사용하지 않습니다. 오직 유저가 원해서 보는 **보상형 비디오(Rewarded Video)**만 탑재합니다.

* **Use Case 1: [Emergency Cool-down]**
    * *Situation:* 무료 유저가 5판을 연속으로 져서 분해하고 있을 때, "과부하"가 걸림.
    * *Action:* "광고를 보고 시스템을 냉각하고 즉시 복수하세요."
* **Use Case 2: [One-time Insight]**
    * *Situation:* 유저가 게임에서 졌는데 왜 졌는지 너무 궁금함. 결제하긴 싫음.
    * *Action:* "광고를 1편 보고 이번 판만 AI의 분석 로그를 훔쳐보세요." (맛보기 제공 -> 결제 유도).

### B. In-App Purchase (IAP - 아이템)
소모성 아이템이 아닌 **'확장 모듈'** 개념으로 판매합니다.

* **Module 1: [Persona Chip Set] ($2.99)**
    * 기본 AI는 건조한 말투지만, 칩셋을 구매하면 다양한 성격의 AI와 대결 가능.
    * *The Poet:* 시적인 표현으로 조롱함.
    * *The Drill Sergeant:* 군대 교관처럼 거칠게 훈수함.
    * *The Psychopath:* 소름 끼치게 정중하고 차가운 말투.
* **Module 2: [Theme Packs] ($1.99)**
    * *Retro OS:* 80년대 녹색 터미널 화면.
    * *Cyberpunk:* 네온 글리치 스타일.
    * *Zen Garden:* 동양적인 바둑판 스타일.

## 4. Monetization Loop (구매 유도 시나리오)

유저가 결제를 결심하게 만드는 심리적 흐름(User Journey)입니다.

1.  **Provocation (도발):** 무료 유저가 `Game 3: Sacrifice Tactics`에서 AI의 'Move 37(변칙수)'에 처참하게 패배함.
2.  **Curiosity (호기심):** "도대체 어떻게 알고 저기를 공격한 거지?" 궁금해 미칠 지경임.
3.  **The Wall (장벽):** '분석' 버튼을 누르니 **"Pro Access Only"** 자물쇠가 걸려 있음.
    * *Micro-copy:* "AI의 뇌 구조를 들여다보려면 관리자 권한(Pro)이 필요합니다."
4.  **Trial (맛보기):** "또는 광고를 보고 1회 해독하기."
5.  **Revelation (계시):** 광고를 보고 AI의 논리("당신의 시선이 우측에 쏠린 틈을 타...")를 읽고 감탄함.
6.  **Conversion (전환):** 매번 광고 보기 귀찮음 + AI를 이기기 위해 공부하고 싶음 -> **$9.99 결제.**

## 5. Technical Implementation Guide (For Cursor AI)

개발 시 상업화 모듈을 어떻게 붙일지에 대한 기술 지침입니다.

### A. Unity IAP Setup
* **Package:** `com.unity.purchasing`
* **Store:** Google Play Console & Apple App Store Connect.
* **Code Structure (ProductCatalog):**
    * `product_pro_upgrade` (Non-Consumable)
    * `product_persona_poet` (Non-Consumable)

### B. AdMob Integration
* **Package:** `Google Mobile Ads SDK`
* **Logic:**
    * 오프라인 게임이지만, 광고를 로드할 때만 인터넷 연결 체크.
    * 인터넷이 없으면? -> 광고 표시 안함.

### C. Security (Hack Prevention)
* 서버가 없으므로 로컬 데이터 변조(결제 우회)에 취약할 수 있습니다.
* **Obfuscation (난독화):** `ProUser = true` 같은 단순한 변수는 메모리 에디터로 뚫립니다.
* **Solution:** XOR 연산이나 Hash 값을 사용하여 로컬 변수를 숨기는 `Anti-Cheat Toolkit` (또는 간단한 커스텀 로직)을 적용하라고 Cursor AI에게 지시해야 합니다.
