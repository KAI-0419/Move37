# 게임 사운드 파일들

이 폴더에는 게임에서 사용되는 사운드 효과 파일들이 위치합니다.

## 사운드 파일 목록

다음 OGG 사운드 파일들이 포함되어 있습니다:

- `move.ogg` - 말 이동 사운드 (select_003)
- `capture.ogg` - 말 포획/공격 사운드 (laser5)
- `win.ogg` - 승리 사운드 (powerUp11)
- `lose.ogg` - 패배 사운드 (phaserDown2)
- `draw.ogg` - 무승부 사운드 (tone1)
- `button_click.ogg` - 버튼 클릭 사운드 (click_001)
- `game_start.ogg` - 게임 시작 사운드 (powerUp3)
- `turn_change.ogg` - 턴 변경 사운드 (switch_003)

## 사운드 출처

모든 사운드는 [Kenney.nl](https://kenney.nl)에서 제공되었습니다.

- **Digital Audio Pack**: 사이버펑크/SF 테마 사운드 (레이저, 파워업 등)
- **Interface Sounds Pack**: UI 인터랙션 사운드 (클릭, 스위치 등)

## 라이선스

이 사운드들은 Creative Commons Zero (CC0) 라이선스로 제공되어 상업적 사용이 가능하며 저작권 표시가 필요하지 않습니다.

Credits: [Kenney](https://kenney.nl) - 고품질 무료 게임 에셋 제공

## 기술 사양

- **파일 형식**: OGG Vorbis (웹 및 모바일 호환)
- **크기**: 4-9KB (최적화됨)
- **길이**: 0.5-1.5초 (빠른 피드백)
- **품질**: 고품질 압축, 낮은 지연시간
